How It Works

Each dashboard reads from a structured data file (CSV or JSON) hosted alongside it. Your team updates the data file from Amazon Ads and Seller Central as part of the existing workflow — same copy-paste, same sources. Upload the updated file to GitHub and the dashboard reflects the latest numbers instantly. One source of truth, zero discrepancies.
